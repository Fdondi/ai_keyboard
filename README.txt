Keyboard that at the press of one button

1) Copies any selected text
2) Opens a new tab in the browser
3) Navigates to ChatGPT
4) Enters a custom prompt depending on the button
5) Pastes the copied text

Available prompts:
Pin 2: "Explain this in detail:"
Pin 3: "Summarize to one or two paragraphs:"
Pin 5: "Rewrite this to be more professional:"
Pin 6: "Translate this to English:
Pin 9: "Write a poem about this:"

Tested on Adafruit Feather 32u4 + Windows/Edge.
